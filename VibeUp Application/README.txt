# Full-Stack Social Media App

A full-stack web application that simulates a mini social media platform with user registration, login, posting, and more.

## Technologies

### Frontend
- HTML, CSS, JavaScript
- React
- Bootstrap

### Backend
- Java
- Spring Boot
- Hibernate (ORM)
- REST APIs
- MVC architecture

### Database
- MySQL (managed via phpMyAdmin)

### Build Tool
- Maven

## Features
- User registration and login
- Create and view posts
- Responsive design
- API communication between frontend and backend
- Database persistence with MySQL

## Notes
This is an ongoing personal project. More features and improvements will be added over time.
