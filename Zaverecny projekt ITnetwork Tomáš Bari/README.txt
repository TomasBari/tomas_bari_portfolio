# Insurance App

A basic console-based Java application created as a final project for the programming course at ITnetwork.

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Description
This project simulates a simple insurance application that demonstrates:
- Basic Java syntax and logic
- Use of OOP principles (classes, inheritance, encapsulation)
- User interaction via the console

## Notes
This was the final project of a course and is intended to showcase foundational Java knowledge.
